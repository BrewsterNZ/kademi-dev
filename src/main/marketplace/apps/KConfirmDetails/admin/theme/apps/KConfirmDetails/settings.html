Terms and conditions

$formatter.checkbox("showTC", "showTC", $!page.getAppSetting($app.instanceId, 'showTC')) If ticked, the user is required to accept terms and conditions to submit the form
This will provide a link to a page with the terms and conditions. You need to create this page.
The text of the link to the terms and conditions, eg 'Click here to see competition terms'
$formatter.checkbox("showExtraFields", "showExtraFields", $!page.getAppSetting($app.instanceId, 'showExtraFields')) If checked, custom fields from the user's group will be shown on the form
$formatter.checkbox("autoreload", "autoreload", $!page.getAppSetting($app.instanceId, 'autoreload')) If checked, the page will be reloaded after the user submits. Recommendation: ticked
$formatter.checkbox("preventClose", "preventClose", $!page.getAppSetting($app.instanceId, 'preventClose')) If checked, the user cant close the modal, so they must submit the form to use the site. Recommendation: ticked
