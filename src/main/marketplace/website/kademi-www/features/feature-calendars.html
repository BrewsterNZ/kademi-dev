Keep track with calendars

Integrate calendars to deliver reminders, events and to-dos...

Create and share

Calendars are a great way to integrate events, to-do lists and reminders into the projects you create using Kademi.

Creative multiple calendars with a different set of events. Target calendars to different groups of users and deliver calendar content across a range of devices.

Confirmation & reminders

Setup a series of email reminders that automatically fire a period time prior to the event date.

Event details

Easily configure your event title & description, date ranges and time-zones.

Export attendees

Keep track of attendees by downloading the latest list of accepted invites via an exported CSV.

Accepting events

Create and send group emails to invite event recipients to attend via an event web-page. Easily customise the event page allowing recipients to accept, decline and invite guests.

CalDav sync

Calendar data can be easily synced with any users mobile device using a CalDav App.
