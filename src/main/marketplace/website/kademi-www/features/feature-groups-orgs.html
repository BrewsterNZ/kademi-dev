Targeted with precision

Powerful features to segment users and target content.

Three dimensional segmentation

Kademi takes segmentation beyond static groups by using organisations, allowing for the creation of
complex organisational hierarchies where each organisation can contain any number of groups.

Organisations

Build a hierarchy of organisations to represent geographical regions, departments and stores for example.

Organisation types

Types are linked to organisation to represent roles for example 'business owner' or 'sales assistant'.

Groups

Create groups which can contain user profiles, websites and permissions.

Targeting groups

Groups allow you to segment people in meaningful ways, more importantly Kademi will allow you to target any type of content or communication to a group.

  • Email & SMS
  • Websites
  • Content including blogs, rewards & training programs

Group access

Configure group access where specific users might require mandatory vetting and approval.

Create custom fields

For each group created easily create any number of custom fields.

Roles and access

Set granular permissions to each group to manage how users access the website and tools you build using Kademi.
