Integrated email & SMS management

Create automated & targeted campaigns driven by user interaction.

Send an email or SMS in minutes

Kademi's integrated email and SMS management will allow you to create messages and send to groups of users in no time at all.

Using the Kademi setup wizard, easily step through each section creating your message along the way, then with a single click, send your message.

Target Groups

Easily setup then target email or SMS messages to pre-defined mailing lists including adding rules to exclude specific users.

Add dynamic content

Using Kademi's templating code add a range of dynamic content to your email or SMS body content.

Test with a click

Send a test message to yourself with a single click, or choose specific recipient addresses.

Design it your way

Easily create beautiful emails in minutes using Kademi's integrated content editor.

Choose from a selection of grid layouts, drop in your content and easily style. If you're a HTML coder upload your images to the Kademi server and drop in your awesome code.

Create email & SMS triggers

Create simple to complex automated email & SMS campaigns using automation. Choose from a range of triggers including sending a message based on how users engage with the content you create using Kademi.

Event delays

Choose when emails are sent by setting event delays between triggers including hours, days, weeks, months and years.

Create exceptions

If you're feeling techie, enter expressions using MVEL syntax to only fire specific emails if the evaluation is true.

Manage triggers

View a list of all pending triggers, their status and due date. Even better remove and add triggers for specific users.

Schedule emails

Create any number of scheduled tasks which could include emailing a weekly report to a key stakeholder.

Track every email & SMS sent

Every email or SMS sent and it's status is listed in your history allowing you to track campaigns and report on messages that were successfully sent vs. failed including individual error codes.

Maximise delivery (DKIM, SPF)

Ensuring your emails get through spam filters is critical for any campaign. Kademi supports both SPF and DCIM standards which can be easily configured.
