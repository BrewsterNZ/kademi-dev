Forms made easy!

Easily create forms, segment uses and capture data in minutes.

Capture data into groups

Kademi allows you to capture form data in several ways. The first method using groups is a great way to segment users then automate actions.

Easily create a group then post the form data to the group. This method allows you to automate actions for example send the user an email or SMS, or even award them points after they have completed and submitted the form.

Contact Us App

Another great method to capture form data is using the Kademi Contact Us App. This method is easily implemented by adding a single action to the form which automatically captures user data into a list. Contact Us data can be reviewed, managed and automatically emailed to stakeholders in a business.
